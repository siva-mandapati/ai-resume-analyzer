# 🤖 AI Resume Analyzer

An AI-powered full-stack web application that 
analyzes resumes against ATS standards, provides 
actionable feedback, and helps candidates improve 
their chances of landing their dream job.

## 📸 Features

- ✅ **ATS Score Analysis** — Get a detailed 
  score breakdown across Impact, Formatting, 
  Keywords, and Relevance
- ✅ **Section Completeness Checker** — Know 
  which resume sections are missing
- ✅ **Skill Gap Roadmap** — Step-by-step 
  learning guide for missing skills
- ✅ **JD Match Analysis** — Compare your resume 
  against a job description
- ✅ **AI Bullet Point Improver** — Rewrite weak 
  bullets with strong action verbs
- ✅ **Resume Chat** — Chat with AI about your 
  specific resume
- ✅ **Cover Letter Generator** — Generate 
  personalized cover letters
- ✅ **Multi-Resume Comparison** — Compare 
  different versions of your resume
- ✅ **Score Progression Tracking** — Track 
  improvement over time

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS
- Recharts

**Backend:**
- Node.js
- Express.js
- MongoDB + Mongoose

**AI & Auth:**
- Google Gemini API
- Google OAuth 2.0
- JWT Authentication
